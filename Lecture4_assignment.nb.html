Generating data

R has several functions to generate and sample data. Were going to work with two of them, runif() and rnorm(), which sample a uniform (all values are equally likely) and a Gaussian (normal) distribution.

gaus_data = rnorm(50, mean=0, sd=30)
unif_data = runif(min=0, max=50, 50)

Lets compare the sampled distributions side by side:

par(mfcol=c(1,2)) # Use two plots instead of one
hist(unif_data, main='Uniform distribution',xlab='Value',las=1,bty='l')
hist(gaus_data, main='Gaussian distribution',xlab='Value',las=1,bty='l')

Notice how they are not that closely distinguishable - if we generate a lot more data points the shape will be much clearer:

unif_err = runif(min=0, max=50, 10000) # normal error
gaus_err = rnorm(10000, mean=0, sd=10)
par(mfcol=c(1,2)) # Use two plots instead of one
hist(unif_err, main='Errors from uniform distribution',xlab='Error',las=1,bty='l', ylim=c(0,2000))
hist(gaus_err, main='Errors from Gaussian distribution',xlab='Error',las=1,bty='l', ylim=c(0,2000))

Sampling error

Lets now try to recover the parameters (mean and SD) of a known distribution we previously generated. First, lets generate our sampling distribution with few data points.

gaus_err = rnorm(10, mean=0, sd=10)
mean(gaus_err)
sd(gaus_err)

Notice how these estimates are not very accurate. If we collected more data, however, the accuracy of the estimations will go up:

gaus_err = rnorm(100, mean=0, sd=10)
mean(gaus_err)
sd(gaus_err)

gaus_err = rnorm(100000, mean=0, sd=10)
mean(gaus_err)
sd(gaus_err)

With a large number of estimations, the recovered parameters are very close to the know true ones. This is anoalogous to the situation with true data collection - we are trying to estimate the true population parameterers from incomplete, random samples.

Lets do this in a slightly different way. Instead of trusting the data generation functions in R, imagine we have access to data from the whole population. We can then sample this dataset by selecting subset of points, just as we would in a real experiment. Lets do that multiple times.

data_sim = rnorm(100000, mean=0, sd=10)
nsim = 10
par(mfcol=c(3,3))
hist(data_sim, xlim=c(-50,50), breaks=seq(from=-50,to=50,by=1),col='lightgrey')
for(i in 1:8) {
  data_sim_subset = sample(data_sim,50)
  hist(data_sim_subset, xlim=c(-50,50),ylim=c(0,8), breaks=seq(from=-50,to=50,by=1),col='red')
  arrows(mean(data_sim_subset),0,mean(data_sim_subset),20,col='red')
}

Notice the mean estimates (red vertical lines) jitter around, but overall a 50 datapoints subsample does a pretty good job of estimating the true mean.

Layout and lattice plotting

R has some serious plotting chops. Sometimes this may make your life easier than you expected. Lets try something a bit silly - loading the gambling behavioral dataset and feeding it to plot()

Overkill, but sometimes great for quick exploratory data plotting.

If we wanted to do something a bit more curated, we can play with plot layout. There are two main options: the simplest is par(mfcol=c(n,m)), which will create n x m plot tiles (empty! - will be populated as you call your plot functions) and the more elaborate layout() function, which will allow a lot more fine-tuning. Lets have a quick look:

par(mfcol=c(1,2))
hist(game_data$payoff,xlab='Payoff ($)',las=1,col='lightblue',main='Payoff distribution across rounds')
hist(game_data$rpe,xlab='Reward prediction error ($)',las=1,col='red',main='RPE')

if we wanted these tiled vertically, we would just use par(mfcol=c(1,2)).

Lets try something a little bit more elaborate: a scatterplot with marginal histograms. For this level of control we will need layout().

Since I anticipate this is a plot that I will call many times over the course of my analysis, Im going to encapsulate it in a function called scatterhist().

scatterhist = function(x, y, xlab="", ylab=""){
  zones=matrix(c(2,0,1,3), ncol=2, byrow=TRUE)
  layout(zones, widths=c(4/5,1/5), heights=c(1/5,4/5))
  xhist = hist(x, plot=FALSE)
  yhist = hist(y, plot=FALSE)
  top = max(c(xhist$counts, yhist$counts))
  par(mar=c(3,3,1,1))
  plot(x,y)
  par(mar=c(0,3,1,1))
  barplot(xhist$counts, axes=FALSE, ylim=c(0, top), space=0, col='lightblue')
  par(mar=c(3,0,1,1))
  barplot(yhist$counts, axes=FALSE, xlim=c(0, top), space=0, col='red', horiz=TRUE)
  par(oma=c(3,3,0,0))
  mtext(xlab, side=1, line=1, outer=TRUE, adj=0, 
    at=.8 * (mean(x) - min(x))/(max(x)-min(x)))
  mtext(ylab, side=2, line=1, outer=TRUE, adj=0, 
    at=(.8 * (mean(y) - min(y))/(max(y) - min(y))))
}
